# Project Brief - Valeris Coaching Website

## Overview

**Valeris Coaching Website** - a trilingual (FR/DE/EN) static website serving as a credibility anchor for a premium solo technology advisory practice in Geneva, Switzerland.

| Attribute | Value |
|-----------|-------|
| Owner | Lucas Challamel |
| Domain | valeris.ch (Infomaniak) |
| Stack | Astro + Payload CMS 3.x + Cloudflare Pages |
| Launch target | End April 2026 |
| Monthly cost | ~CHF 1.50 (domain only) |

## Mission

Provide a professional web presence that Swiss enterprise buyers can verify when evaluating Valeris Coaching. The site is a credibility anchor, not a marketing platform or lead generation tool.

## Success Criteria

| Criteria | Target |
|----------|--------|
| Pages | 8 per language (24 total: FR + DE + EN) |
| Performance | Lighthouse 95+ all categories |
| Cost | < CHF 10/month all-in |
| Maintenance | < 8 hours/year |
| Load time | < 1.2s First Contentful Paint |
| Uptime | 99.9% (Cloudflare SLA) |

## Scope

### In Scope (MVP - Phase 0)

- 8-page trilingual static site (FR primary, DE secondary, EN tertiary)
- Payload CMS for content management
- Cloudflare Pages hosting
- Cal.com booking integration
- Privacy-respecting analytics
- Swiss DPA (nDSG) compliant legal pages

### Out of Scope (Phase 2+)

- Blog / insights section (until 3+ quality articles exist)
- Case studies (until real client engagements complete)
- Client testimonials (until permission obtained)
- Contact form (booking widget replaces this)
- Newsletter signup
- E-commerce / online payments

## Constraints

- Solo practitioner: max 50 billable days/year, website must not be a maintenance burden
- No client testimonials or case studies at launch (pre-client phase)
- Must respect Swiss DPA (nDSG) for privacy
- French is primary language (Romandie market), German for Swiss-German reach, English for international
- Premium positioning: design must signal quality, not startup

## Key Stakeholders

| Stakeholder | Role |
|-------------|------|
| Lucas Challamel | Owner, developer, content author |
| Target buyers | Swiss CTOs, VPs Engineering, CISOs, HR leaders |
